# PantryPal
WELCOME TO PANTRYPAL
## Usage:
note: you need to have an openapi api key.

or else the program won't function properly.
### build the application
linux/macos
```Shell
gradle build
```
windows
```Batchfile
./gradlew build
```

### run the application

linux/macos
```Shell
gradle run --args="<OPENAPI_API_KEY>"
```
windows
```Batchfile
./gradlew run --args="<OPENAPI_API_KEY>"
```
